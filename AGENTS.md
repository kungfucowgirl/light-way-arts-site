# Light Way Arts Website Instructions

## Project goal
Build a clean, elegant, mobile-responsive website for Light Way Arts Foundation, an Austin-based nonprofit arts foundation focused on cross-cultural storytelling, bilingual arts education, performance, shadow puppetry, family-centered arts, and Chinese/American cultural connection.

## Source of truth
The `/content` folder contains the approved project context. Read those files before making copy, design, or structure decisions.

Do not invent:
- IRS status details
- donation processing details
- confirmed sponsors
- event dates
- board/member details
- financial claims
- grant claims

Use placeholders when information is missing.

## Tone
Warm, credible, artistic, sincere, clear, nonprofit-friendly.

Avoid:
- corporate buzzwords
- overly generic AI language
- exaggerated claims
- "changing the world" clichés
- dense arts jargon

## Design
Preferred feeling:
- warm gold
- deep brown
- burgundy
- cream/off-white
- light/path imagery
- subtle Chinese calligraphy or ink influence
- simple, flat, elegant nonprofit visual language

Avoid:
- cheesy East Asian decorative motifs
- red/gold restaurant-style branding
- overly techy SaaS design
- cluttered layouts

## Technical
Prioritize:
- simple code
- accessibility
- fast loading
- mobile responsiveness
- easy deployment to Netlify or Vercel

Use plain HTML/CSS/JavaScript for the first version unless there is a strong reason to use React.

## Review checklist
Before finishing:
- Confirm the site runs locally.
- Confirm layout works on mobile and desktop.
- Confirm copy matches `/content`.
- Confirm there are no fake claims.
- Update README with setup and deployment instructions.
